# Jacob Orbach – Portfolio

A modern, responsive portfolio website showcasing my work as a full-stack developer and SaaS builder.

## Overview

This portfolio highlights my experience in building fast, lightweight digital products, from ultra-optimized SaaS tools to large-scale WordPress ecosystems. The site features a clean, modern design with smooth animations and a mobile-responsive layout.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Smooth Scrolling**: Native smooth scrolling navigation with section highlighting
- **Mobile Menu**: Hamburger menu for mobile navigation
- **Fade-in Animations**: Scroll-triggered animations for engaging user experience
- **Featured Projects**: Showcase of SaaS products and projects with detailed descriptions
- **Contact Integration**: Direct email and LinkedIn contact links

## Getting Started

### Viewing the Portfolio

Visit the live website: **[jacoborbach.com](https://jacoborbach.com)**

### Local Development

To run locally, simply open `index.html` in a web browser. No build process or server required.

```bash
# Option 1: Open directly
open index.html

# Option 2: Use a local server (recommended for development)
python3 -m http.server 8000
# Then visit http://localhost:8000
```
